1. Сделайте упрощённый вариант канбан-доски, на которой можно:
- добавить\редактировать\удалить карточки;
- добавить\редактировать\удалить колонки;
- перетаскивать карточки между колонками.
2. Добейтесь того, чтобы решение было удобным, быстрым, аккуратным. Подумайте, как сделать так, чтобы пользователю было комфортно и приятно пользоваться вашим продуктом. Посмотрите как сделан UI в Trello.
3.Макет по ссылке: https://www.figma.com/file/odIzynKOUdBfpHccxxixrK/Kanban-(Copy)
4. Использовать в коде минимальное количество внешних зависимостей. Функционал должен быть написан на чистом JavaScript ES6, без использования jQuery и разнообразных фреймворков. Допускается использовать библиотеки для перетаскивания карточек.
5. Все кнопки\карточки\элементы должны иметь логичные визуальные эффекты (например ховеры на кнопках, анимации перетаскивания).
6. Настроить рабочее окружение используя webpack:
- Конвертацию SCSS в css
- Минификацию css
- Минификация js
- Оптимизация изображений
- Настроить локальный dev сервер и watcher для него на файлы проекта
- Вынести команды для dev и production билда в главный package.js

7. Задача со звездочкой: Добавьте авторизацию и хранение данных при помощи Firebase.
8. Задача со звездочкой 2: Используйте Github Pages или netlify для демонстрации полученного результата.
